作用

在文档 Documentation/kernel-parameters.txt 中说明了 initcall_debug 命令参数的作用: 

initcall_debug	[KNL] Trace initcalls as they are executed.  Useful
    for working out where the kernel is dying during
    startup.

简单来说就是可以展示初始化函数的执行。
